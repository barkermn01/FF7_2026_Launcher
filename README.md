# FFVII Launcher Source

This is a working source build decompiled from the debug version of the launcher shipped with the 2026 Steam Release of Final Fantasy VII.

## Origin

This source code was decompiled using dotPeek & ILSpy from the original FFVII_LAUNCHER.exe included with the Steam release. No modifications have been made to the functionality - this is purely a buildable version of the original launcher.

## Building

Run `build.bat` to compile using MSBuild from Visual Studio 2026.
